## Features

- **Console** ‚Äì Log, warn, error, info, and execute JavaScript.
- **Network** ‚Äì Monitor XHR, Fetch, and WebSocket requests with headers, payload, and responses.
- **Sources** ‚Äì View HTML, CSS, and JS files loaded on the page.
- **Cookies** ‚Äì View all cookies stored on the current page.
- **Panel** ‚Äì Close the panel, reopen it with a draggable gear icon.
