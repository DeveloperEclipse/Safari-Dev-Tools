## Features

- **Console** â€“ Log, warn, error, info, and execute JavaScript.
- **Elements** - View HTML elements on the current page.
- **Network** â€“ Monitor XHR, Fetch, and WebSocket requests with headers, payload, and responses.
- **Sources** â€“ View HTML, CSS, and JS files loaded on the page.
- **Cookies** â€“ View all cookies stored on the current page.
- **Panel** â€“ Close the panel, reopen it with a draggable gear icon.
